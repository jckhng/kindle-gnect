# Exact Four in a Row Extension Files

Copy these files into your Kindle extension folder:

- `config.xml` -> `/mnt/us/extensions/exact-four-in-a-row/config.xml`
- `menu.json` -> `/mnt/us/extensions/exact-four-in-a-row/menu.json`
- `launch_exactfourinarow.sh` -> `/mnt/us/extensions/exact-four-in-a-row/launch_exactfourinarow.sh`
- `stop_exactfourinarow.sh` -> `/mnt/us/extensions/exact-four-in-a-row/stop_exactfourinarow.sh`
- `tail_log_exactfourinarow.sh` -> `/mnt/us/extensions/exact-four-in-a-row/tail_log_exactfourinarow.sh`

Optional document shortcut:

- `shortcut_exactfourinarow.sh` -> `/mnt/us/documents/shortcut_exactfourinarow.sh`

KUAL is the reliable tap-launch path. The stock Kindle home screen normally
does not execute `.sh` files directly.
