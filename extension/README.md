# Kindle Gnect Extension Files

Copy these files into your Kindle extension folder:

- `config.xml` -> `/mnt/us/extensions/kindle-gnect/config.xml`
- `menu.json` -> `/mnt/us/extensions/kindle-gnect/menu.json`
- `launch_kindlegnect.sh` -> `/mnt/us/extensions/kindle-gnect/launch_kindlegnect.sh`
- `stop_kindlegnect.sh` -> `/mnt/us/extensions/kindle-gnect/stop_kindlegnect.sh`
- `tail_log_kindlegnect.sh` -> `/mnt/us/extensions/kindle-gnect/tail_log_kindlegnect.sh`

Optional document shortcut:

- `shortcut_kindlegnect.sh` -> `/mnt/us/documents/shortcut_kindlegnect.sh`

KUAL is the reliable tap-launch path. The stock Kindle home screen normally
does not execute `.sh` files directly.
